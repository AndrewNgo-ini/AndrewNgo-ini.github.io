# Install
macos: brew install hugo
linux: sudo snap install hugo

# Commands
hugo new site "name"
hugo new posts/my-first-post.md

# Build
hugo -D

# Start server
hugo server 

# Notes:
remember to clone submodule or else local won't be rendered
